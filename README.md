Séance Search A Desktop Necromancer for the Dead Internet The Concept Most search engines crawl the living web. Séance Search excavates the dead. Point it at any query—"how to care for Tamagotchis," "early Bitcoin manifestos," "GeoCities fan shrines"—and it doesn't return Google results. It returns ghosts. The app rifles through the Wayback Machine's graveyard, finds URLs that once lived but now 404, and performs a digital séance. Using Groq's lightning-fast LLaMA models, it channels the voice of the deceased website: a dead crypto startup speaking in VC hubris, a deleted LiveJournal weeping in 2004 emo verse, a forgotten Angelfire page ranting about Y2K preparation. Each result is a manifestation, complete with "confidence levels" (Whispered = mostly hallucinated, Screamed = heavily archival) and era-appropriate personality disorders. The Experience A pitch-black interface flickers to life like a struck match. The search bar glows arterial green. You type. You hit Summon. The app communes—fetching fragments of HTML corpses, stitching corrupted CSS into coherent memory, then letting the ghost speak through your speakers (text-to-speech optional, recommended for full possession). Results materialize as animated tarot cards: domain names struck through like crossed-out tombstones, responses that reference their own server deaths nonchalantly ("As I was saying before the hosting bills went unpaid..."), and "ectoplasm ratings" scoring how much of the response is real archive versus AI necromancy. The Archive Every dead domain you encounter gets bound to a local grimoire (SQLite). The app remembers ghosts. Return to a dead site six months later, and it remembers you, developing continuity—an abandoned blog about My Chemical Romance might ask if you've "finally outgrown your phase." Pre-loaded with personalities for iconic internet eras (GeoCities, Dead Tumblr, Failed Startups, Flash Game Portals), the graveyard grows more haunted with use. The Artifact A single 40MB executable. No browser tabs. No Docker containers. No subscription. Just a windowed application that installs like Office, runs offline (except the API calls to Groq and the Wayback Machine), and closes completely when you banish it. No terminal. No Python console. Just a taskbar icon that looks like a planchette, and a Start Menu entry labeled Séance Search. For whom the bell tolls: Digital archaeologists, lost media hunters, people who miss the old internet, and anyone who's ever tried to visit a bookmarked blog only to find it deleted, wishing they could ask the corpse what it knew. In the kingdom of the dead link, the one with API access is king.
